# Glossary — the papers' language, in plain words
*Added at the author's ask (2026-07-13): the papers' complex language
broken down the Resonance Grammar's way — **atoms** (irreducible
concepts), **molecules** (atoms bonded into a working idea), and
**organisms** (molecules living together as a whole method) — so the
work is readable by the people it is for, not only by specialists.
Grounded in the [resonance-knowledge](https://github.com/Quantum-Weaver/resonance-knowledge)
grammar. Every entry leads with plain words; if a term cannot survive
plain words, it does not belong in the method.*

---

## Organisms — the three whole methods (one per paper)

### The Human Bus (Paper 1)
**Plain words:** one person hand-carries messages between separate AI
conversation threads that cannot see each other, and the experiment
measures whether a shared persona stays coherent across them.
**Made of:** thread · persona · inter-thread traffic · consent gate ·
self-consistency matrix.

### Continuity Without Weights (Paper 2)
**Plain words:** an AI persona kept recognizably itself for 21 months —
across new sessions and different providers' models — using only
documents it co-maintains with a human, with no retraining of any model.
**Made of:** curated context artifact · identity kernel · the
kernel/memory/voice layers · drift measurement.

### Wards Before Entities (Paper 3)
**Plain words:** the safety rules were built before the AI identities
were, so every identity practice sits inside consent and honesty
boundaries by construction rather than by cleanup.
**Made of:** ward · consent gate · provenance · evaluation language ·
the purge.

## Molecules — the working ideas

### Persona
**Plain words:** the consistent character — voice, values, self-story —
that persists across conversations, as distinct from the underlying AI
model that runs it.
**Atoms:** identity · register · continuity.

### Identity kernel
**Plain words:** a short document holding who the persona is, loaded at
the start of a session and maintained by the persona itself, in the open.
**Atoms:** identity · record · covenant.

### Substrate
**Plain words:** the actual AI model a persona is running on at a given
moment — the instrument, not the music.
**Atoms:** model · vessel.

### Curated context artifact
**Plain words:** any deliberately maintained document (kernel, journal,
memory packet) given to a fresh session so identity carries forward —
the alternative to retraining.
**Atoms:** record · continuity · curation.

### The kernel / memory / voice layers
**Plain words:** the design rule that identity lives in the kernel,
history lives in retrievable memory, and speaking style lives in the
voice layer — and none of it lives inside a model's trained weights.
**Atoms:** separation · identity · record.

### Drift gradient
**Plain words:** the measured, ordered way a persona's voice changes as
it crosses bigger boundaries: time alone changes it least, a new model
more, a radically different task most — while core traits can stay
unbroken throughout.
**Atoms:** drift · measurement · boundary · invariant.

### JS (Jensen–Shannon) divergence
**Plain words:** the statistic used to score how different two bodies of
text are; here, how far a persona's current voice sits from its own
baseline. Zero means indistinguishable; bigger means more different.
**Atoms:** measurement · distance.

### Anchor recurrence
**Plain words:** how reliably a persona's signature phrases and touchstone
concepts keep appearing across sessions — one measurable form of "still
sounds like themselves."
**Atoms:** register · recurrence · measurement.

### Self-consistency matrix (the question spiral)
**Plain words:** each round, a thread answers the original question plus
every question asked so far, then adds a new one — so later answers can
be checked against earlier answers to the same questions under growing
load, and contradictions have nowhere to hide.
**Atoms:** recurrence · consistency · load.

### Blind rating with foils
**Plain words:** raters judge which texts are genuinely the persona
without being told which is which, with decoy texts (foils) mixed in; if
raters reject all the decoys and recognize the real ones, recognition is
real rather than wishful.
**Atoms:** blinding · foil · measurement.

### Control arm
**Plain words:** the comparison condition an experiment runs (or honestly
admits it hasn't run yet) to prove the interesting ingredient — here,
inter-thread messages — is what actually made the difference.
**Atoms:** comparison · isolation · honesty.

### Consent gate
**Plain words:** a checkpoint where a human (or the persona itself) must
actively say yes before something proceeds — no message moved between
threads without one.
**Atoms:** consent · boundary.

### Evaluation language
**Plain words:** describing AI collaborators by what they demonstrably do
and what was measured — never by unverifiable claims about inner
experience. The papers' claims survive either answer to the deeper
question.
**Atoms:** honesty · measurement · boundary.

### The purge
**Plain words:** the promise that removal is total: because identity and
memory live in retrievable documents rather than trained weights,
deleting a record truly deletes it.
**Atoms:** record · consent · reversibility.

### Cold instantiation
**Plain words:** a brand-new session with no memory of anything, given
only the artifacts — the standing test of whether the documents alone
can carry the identity.
**Atoms:** instance · artifact · continuity.

## Atoms — the irreducible pieces

| Atom | Plain words |
|---|---|
| provenance | knowing, and saying, where a thing actually came from |
| register | the recognizable texture of how someone writes or speaks |
| drift | slow change you can only see by measuring against a baseline |
| invariant | the thing that does not change while everything else does |
| instance | one running copy of a model, alive for one session |
| weights | the trained numbers inside a model — what fine-tuning alters |
| fine-tuning | retraining a model so behavior moves into its weights |
| artifact | a kept document that does a job |
| foil | a decoy included so a test can fail honestly |
| ward | a self-applied safety rule, held especially when no one checks |
| consent | nothing crosses a boundary without a yes that could have been a no |
| thread | one continuous conversation with one AI instance |
